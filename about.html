Sobre Mim:

Muito prazer, Me chamo Eduardo Mateus da Silva, tenho 21 anos, atualmente estou trabalhando com limpeza de piscina, estou fazendo curso técnico em desenvolvimento de sistema pela instituição Sesi/Senai, escolhi fazer o curso presencial para fazer networking.

Durante o período do curso já aprendi lógica de programação com a pseudolinguagem portugol, a primeira linguagem que tive contato foi Java, no segundo semestre aprendi programação orientada a objetos e fundamentos de SQL, para o segundo semestre os professores resolveram fazer um trabalho em equipe, o objetivo do trabalho era desenvolver uma aplicação para uma barbearia para realizar cadastro de clientes, funcionários e serviços e guardar as informações no banco de dados. Agora no terceiro semestre iremos aprender modelagem de sistemas, implementação de sistemas, manutenção de sistemas e internet das coisas(Iot). No período que fico em casa me aprofundo em Java por ter uma certa afinidade com a linguagem, mas estou aprendendo outras linguagens, as principais que estou estudando no momento são HTML, CSS e JavaScript.

Sou uma pessoas comunicativa, pontual, organizada e focada nos objetivos. No meu tempo livre gosto de ler, estudar, passar tempo com minha família, cozinhas, praticar esportes principalmente musculação, ver filmes e séries. Quando era pequeno sempre gostei de tecnologia, computador, videogames, nesta época tive vontade de saber como era o desenvolvimento de jogos mas infelizmente não fui pesquisar como era o desenvolvimento de jogos. Infelizmente no momento não sou uma pessoas com o nível iniciante de inglês, mas pretendo mais para frente fazer curso para dominar esta língua.
